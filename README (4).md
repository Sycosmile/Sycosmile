### What I actually do

I look for the gap between what a system is supposed to do and what it actually lets you do. Sometimes that's a smart contract that trusts an oracle it shouldn't. Sometimes it's a session that never really ends. Either way, same instinct: find the assumption, break it, write it up.

Right now that means running two tracks in parallel:

- **Smart contract security** — live work across Immunefi, CodeHawks, Sherlock, Cantina, and OnlyDust
- **Classic web app bounty hunting** — building this track deliberately from the ground up: OWASP Top 10, auth/session internals (JWT, OAuth), API security, the whole recon-to-report pipeline
